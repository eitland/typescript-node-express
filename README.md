# typescript-node-express

A minimal Node.js/Express app written in TypeScript.

## Quickstart

Install with

```
npm install
```

and run with

```
npm start
```

Then open your browser and load `http://localhost:8080/`.

To get rid of generated files, run

```
npm run clean
```
